# عروج AI — تحليل استراتيجي متكامل لمشروعك

عروج AI هو أداة تحليل استراتيجي متطورة مصممة لمساعدة رواد الأعمال وأصحاب المشاريع على فهم نقاط القوة والضعف في مشاريعهم. من خلال الإجابة على 25 سؤالاً مفصلاً، يقوم النظام بإنشاء خارطة طريق شاملة وتوصيات عملية باستخدام الذكاء الاصطناعي.

## المميزات الرئيسية
- **تحليل عميق:** يغطي 5 مجالات أساسية (الأساسيات، الهوية، المبيعات، التسويق، الطموحات).
- **ذكاء اصطناعي احترافي:** يستخدم نماذج لغوية متطورة لتقديم استشارات مخصصة.
- **واجهة مستخدم عصرية:** تصميم متجاوب، جذاب، وسهل الاستخدام باللغة العربية.
- **تقارير فورية:** احصل على نتيجة جاهزية مشروعك وتقرير مفصل في ثوانٍ.

## التقنيات المستخدمة
- **HTML5 & CSS3:** لهيكل الصفحة والتصميم المتقدم (بما في ذلك المتغيرات والأنماط المتجاوبة).
- **JavaScript (Vanilla):** لإدارة منطق النموذج، التنقل بين الخطوات، والتفاعل مع API.
- **AI Integration:** ربط مع نماذج الذكاء الاصطناعي لتقديم التحليلات.

## كيفية الاستخدام
1. قم بفتح ملف `index.html` في متصفحك.
2. انقر على "ابدأ التحليل الآن".
3. أجب على الأسئلة الـ 25 بدقة.
4. انتظر ثوانٍ حتى يقوم النظام بتحليل إجاباتك.
5. استعرض تقريرك الاستراتيجي وتواصل مع "عروج" للتنفيذ.

## ملاحظات أمنية
- تم تحسين الكود لضمان أفضل أداء.
- **تنبيه:** يوصى بنقل مفاتيح الـ API إلى خادم خلفي (Backend) في بيئة الإنتاج لضمان الأمان.

## التواصل
للمزيد من المعلومات أو الدعم، يمكنك التواصل مع وكالة عروج عبر الموقع الرسمي أو واتساب.
